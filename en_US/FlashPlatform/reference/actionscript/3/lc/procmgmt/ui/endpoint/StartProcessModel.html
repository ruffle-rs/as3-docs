The StartProcessModel class acts as the model for the StartProcess component.

The StartProcessModel class loads the contents of the list data into the navigation area in the StartProcess component. The list data contains favorites and the available categories configured on a LiveCycle ES2 server. Depending on the selection of a Category or FavoriteCategory object, the appropriate set of forms to start a process are loaded into the content area of the component. Each form that starts a process is represented as an Endpoint object.
