Packageflashx.textLayout.elements
Classpublic class ListElement
InheritanceListElement ContainerFormattedElement ParagraphFormattedElement FlowGroupElement FlowElement Object

Language Version: ActionScript 3.0
Runtime Versions: Flash Player 10, AIR 1.5

The List class is used for grouping together items into a numbered or unnumbered list. A ListElement's children may be of type ListItemElement, ListElement, ParagraphElement, or DivElement.

Each ListElement creates a scope with an implicit counter 'ordered'.

Default MXML PropertymxmlChildren
