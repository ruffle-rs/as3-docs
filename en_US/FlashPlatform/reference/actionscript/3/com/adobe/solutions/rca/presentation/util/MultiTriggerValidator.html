Classpublic class MultiTriggerValidator
InheritanceMultiTriggerValidator Validator EventDispatcher Object
Subclasses EndDateOrderValidator, StartDateOrderValidator

Language Version: ActionScript 3.0
Product Version: Review, Commenting, and Approval Building Block 10.0
Runtime Versions: Flash Player 10.2, AIR (unsupported)

An extension of Flex Validator class that triggers the validation on more than one configurable event names. The default Flex Validator class allows you to configure the triggerEvent. User can specify only one event name. This class allows you to provide an array of event names each of which triggers the validation.
